# 🍔 SnackQ - Smart Canteen Order & Queue System

SnackQ is a web-based Smart Canteen Order & Queue Management System developed using Flask and SQLite. The system allows customers to place food orders online, monitor queue status, and reduce waiting time. Staff can efficiently manage orders while the admin oversees users, menu items, sales, and reports.

---

## Features

### Customer
- Register/Login
- Browse Menu
- Search & Filter Food Items
- Add Items to Cart
- Place Orders
- Track Order Status
- View Order History
- Reorder Previous Orders
- Submit Feedback

### Staff
- Login
- View Incoming Orders
- Update Order Status
- Manage Queue

### Admin
- Dashboard
- Manage Customers
- Manage Staff
- Manage Menu
- View Sales Report
- Analytics Dashboard
- Generate PDF Reports
- View Feedback
